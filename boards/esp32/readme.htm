ESP32 with HomeDing

The HomeDing library is compatible to the ESP32 Arduino environment.

Specific to ESP32 are the following features:

  • In contrast to the ESP8266 there are no other names for the GPIO pins like the one from NodeMCU. The numbers in the processor datasheet are used.

  • The Touch Element allows using the ESP32 built-in touch features on the GPIO pins.

  • The MY9291 Element only works with ESP32 by the used library.
