Pages marked with a Sensor tag

Analog Element -- The AnalogElement is used to capture the voltage level from the Analog Input pin with the integrated ADC and to emit corresponding events. It also handles some of the often occurring problems around capturing analog values.

DHT Element -- The DHTElement allows retrieving temperature and humidity values from the DHT family and AM2320 sensors and creates actions when new values are available.

The BMP280 Element -- The BMP280 Element supports the BMP280 sensor chips for air temperature and air absolute barometric pressure.
