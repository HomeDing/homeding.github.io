Web Manifest for HomeDing based devices

The Web files for the homeding devices has a manifest file site.webmanifest. It supports using the UI of a HomeDing device as a Progressive Web Application e.g. that can be installed as an icon on the desktops

This manifest file is a JSON document that contains startup parameters and application defaults for when a web application is launched.
